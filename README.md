# Banking_EDA-Data-Analyst-Project-
This project performs an analytics workflow on banking data: from data ingestion and processing, to exploratory data analysis (EDA). The goal is to derive actionable insights for banking operations using real-world data.

The project provides actionable insights into customer behavior, enabling better risk assessment, improved marketing personalization, and optimized financial decision-making. It demonstrates proficiency in real-world EDA, data visualization, and extracting meaningful business insights from raw data.
